You can upload a new question list here, the file must be a .xlsx table.

The columns must contain 'Question','Answer'. (It can also contain other columns but Question and Answer are required)

'Status' is an optional column. If you set it, the script will decide the status of the question.


There are three types of upload you can choose: Append, Overwrite and Clear & Overwrite

If you choose Append, the uploaded questions will be appended to existing database. Duplicate questions may exist and you will be warned.

If you choose Overwrite, when duplicated questions are found, they will be replaced by uploaded ones. Other questions will be appended to existing database.

If you choose Clear & Overwrite, all the existing questions will be deleted and only the uploaded ones will be kept.

Please note that if you search for a question which is duplicated, only the first-imported one will be shown.


And, by the way, Excel doesn't support multiple lines in one cell, so if you want to write question / answer with multiple lines, use \n as line break.
