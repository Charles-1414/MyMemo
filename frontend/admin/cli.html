Command List

User Management

1. get_user_info [userId] - Get detailed user info of [userId]

2. create_user [username] [email] [password] - Create a user by admin, user will be shown as invited by this admin

3. delete_user [userId] - The account must be marked as deletion first, and admin will be able to bring the deletion schedule forward

4. set_privilege [userId] [item] [value] - Add [item] privilege for user [userId] ([item] can be question_limit, book_limit, allow_group_creation, group_member_limit) | If privilege exists, then update it

5. remove_privilege [userId] [item] - Delete [item] privilege from user [userId]

6. set_name_tag [userId] [tag] [tag type] - Add name tag [tag] for user [userId] ([tag type] is the color of tag)

7. remove_name_tag [userId] - Remove nametag from [userId]

8. ban_account [userId] - Ban account

9. unban_account [userId] - Unban account

10. get_user_count - Get user count
