Admin Command Line Interface

Command

Command List

1. get_user_info [userId] - Get detailed user info of [userId]

2. create_user [username] [email] [password] - Create a user by admin, user will be shown as invited by this admin

3. delete_user [userId] - The account must be marked as deletion first, and admin will be able to bring the deletion schedule forward

4. delete_user_completely [userId] - Wipe all the data of [userId] (this should be requested by the user)

5. mute [userId] [duration] - Mute [userId] for [duration] days | To mute forever, set [duration] to -1

6. unmute [userId] - Unmute [userId]

7. set_privilege [userId] [item] [value] - Add [item] privilege for user [userId] ([item] can be question_limit, book_limit, allow_group_creation, group_member_limit) | If privilege exists, then update it

8. remove_privilege [userId] [item] - Delete [item] privilege from user [userId]

9. set_name_tag [userId] [tag] [tag type] - Add name tag [tag] for user [userId] ([tag type] is the color of tag)

10. remove_name_tag [userId] - Remove nametag from [userId]

11. ban [userId] [reason] - Ban account

12. unban [userId] - Unban account

13. get_user_count - Get user count

15. revert_email_change [userId] [old email]\nHelp user revert their email to [old email] only if they have used that old email.

Note that you should let user send an email with old email to you for verification.

15. restart - Restart backend server
