You can export the data here.

There are two options for you to choose: Excel Table and Raw Database File

If you choose Excel Table, you will get all the words, definitions and status in a .xlsx file which is human-readable.

If you choose Raw Database File, the server will directly send you the server database file after removing some sensitive data (such as your import password).

Please note that the words and definitions stored in the raw database file are encoded with base64. Status code are integars: {1: "Default", 2: "Tagged", 3: "Removed"}

And by the way, for server safety, you are not allowed to upload database directly. If you want to do so, please upload the file using console.


To prevent abusing, you must enter the import password.
